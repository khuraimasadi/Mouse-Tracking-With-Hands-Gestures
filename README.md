# Mouse-Tracking-With-Hands-Gestures
By running this file your camera will be automatically opened and then you can control your cursor on screen using hands gestures. You can change cursor position on screen by moving index finger and clicked by minimizing the distance between index and middle finger. You change variable ws and wh according to your own screen size.
